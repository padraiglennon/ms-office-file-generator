# Common File Generator

Create realistic test files - PowerPoint, Word, Excel, PDF, and Markdown - either
from scratch at a chosen complexity, or by **filling** a ready-made "Golden"
template with your own data. Use it from the command line, the web UI, or the
JSON API.

When the web service is running, an interactive API reference is served at
`/docs` (Swagger) and `/redoc`.

## Guides

- [Preparing a Golden template](golden-file-guide.md) - marking up the text,

- [The data file (JSON), field by field](config-guide.md) - what each field means
  and how to write your own.
